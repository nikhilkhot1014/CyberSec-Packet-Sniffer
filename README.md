# CyberSec Packet Sniffer

A Python-based GUI Packet Sniffer built using Tkinter and Scapy for real-time network traffic monitoring and ARP spoofing detection.

## Features

- Real-time packet sniffing
- Displays source/destination IP and MAC addresses
- Detects TCP, UDP, IP and ARP packets
- ARP spoofing detection alerts
- Export logs to CSV
- Hacker-style Tkinter GUI

## Installation

```bash
pip install -r requirements.txt
python main.py
```

> On Windows, install Npcap and run the application as Administrator.
